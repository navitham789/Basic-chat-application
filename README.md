# Basic Chat Application (Real-Time)

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Socket.io
- Database: MongoDB

## Features
- Real-time messaging
- Socket rooms
- Persistent messages
- User online/offline tracking
- Timestamped messages
- Separate socket and API logic

## Architecture
- Socket layer for real-time events
- API layer for REST operations
- Scalable design
